## Development

### Prepare Dependencies

```bash
go get
go install
```

### Run

```bash
go run server.go
```

### Build

```bash
go build
```

### Test

```bash
go test ./...
```

### Test and Check Coverage

```test
go test ./... -covermode=atomic -coverprofile cover.out
```
